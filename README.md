> Windows 11 스타일의 가볍고 빠른 마크다운 메모 앱

HwanNote는 Windows 11 메모장에서 영감을 받은 데스크톱 마크다운 메모 앱입니다. 토글 블록, 체크리스트, 표 등 다양한 편집 기능을 지원하며, 작성한 메모는 표준 마크다운(`.md`) 파일로 저장되어 다른 에디터에서도 열 수 있습니다.

---

## 다운로드

[GitHub Releases](../../releases)에서 최신 버전을 다운로드하세요.

설치 파일(`.exe`)을 실행하면 바로 사용할 수 있습니다.

---

## 주요 기능

### 편집

- 제목(H1~H3), 굵게, 기울임 서식
- **토글/접기 블록** — Notion 스타일의 접이식 콘텐츠
- **체크리스트** — 중첩 지원, 체크 상태 저장
- **표** — 행, 열, 헤더 지원
- **링크** — URL 자동 감지, 붙여넣기 시 자동 링크
- 글머리 기호 및 번호 목록
- `.txt` 파일 가져오기

### 탭

- 여러 메모를 동시에 열어두고 전환
- 드래그 앤 드롭으로 탭 순서 변경
- 탭 고정/해제
- 저장되지 않은 변경 표시

### 사이드바

- 계층형 폴더 관리 — 생성, 이름 변경, 삭제
- 제목 + 내용 통합 검색
- 해시태그 자동 추출 (`#태그`)
- 정렬 — 최근 수정순, 이름순, 생성일순

### 외관

- **다크 / 라이트 / 시스템** 테마
- 줄 간격 조절
- 한국어, English 지원

### 파일 관리

- **자동 저장** — 변경 후 자동으로 저장
- 마크다운(`.md`) 파일로 로컬 저장
- `.md` 파일 연결 — 마크다운 파일을 HwanNote로 바로 열기
- 자동 업데이트 지원

## 키보드 단축키

모든 단축키는 설정에서 변경할 수 있습니다.

| 기능 | 기본 단축키 |
|---|---|
| 새 메모 | `Ctrl+N` |
| 메모 저장 | `Ctrl+S` |
| 사이드바 토글 | `Ctrl+B` |
| 다음 탭 | `Ctrl+Tab` |
| 이전 탭 | `Ctrl+Shift+Tab` |
| 탭 닫기 | `Ctrl+W` |
| 굵게 | `Ctrl+B` (에디터) |
| 기울임 | `Ctrl+I` (에디터) |
| 체크리스트 | `Ctrl+Shift+X` (에디터) |
| 토글 블록 | `Ctrl+Shift+T` (에디터) |

---

## 저장 위치

메모는 `문서/HwanNote/Notes/` 폴더에 표준 마크다운 파일로 저장됩니다. 설정에서 저장 경로를 변경할 수 있습니다.

---

## 알려진 제한사항

- Windows 10/11(64-bit) 전용
- 클라우드 동기화 미지원 — 모든 메모는 로컬에 저장됩니다
- 이미지 삽입 미지원
